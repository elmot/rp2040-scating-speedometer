Speedometer for nordic skating
====

Components: 
---
 * GPS module: NEO-6M with antenna
 * Display: flexible WS2812B panel 8*32
 * MCU: Pimoroni TINY 2040 8Mb. Can be replaced with Raspberry Pi Pico.
 * 1 Cell LiPo battery
 * 1 Shottky diode (virtually any type)
